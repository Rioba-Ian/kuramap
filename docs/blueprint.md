# **App Name**: Mwanzo Vote Locator

## Core Features:

- Voter Center Search: Users can input their location (e.g., constituency, ward, or use GPS) to find nearby registration or polling centers.
- Center Details View: Display detailed information for each identified center, including its official name, physical address, and operating hours.
- Interactive Map Display: Show all relevant registration centers on an interactive map, allowing users to visualize their precise locations relative to their current position.
- Optimal Route Guidance Tool: Utilize AI to recommend the most accessible or convenient registration center based on the user's current location and provide routing options (e.g., walking, public transport).
- First-Time Voter Eligibility Information: Provide clear and concise information regarding the eligibility criteria and required documents for first-time voter registration in Kenya.
- Polling Station Reviews: Allow users to submit comments and reviews about polling stations, indicating issues like closure, long queues, or helpful staff, and view aggregated feedback from other users.

## Style Guidelines:

- Primary color: A rich, deep indigo (#562EB3), conveying trust and authority, without being overly formal. Chosen to be highly versatile.
- Background color: A very light, desaturated lavender (#F2F0F7), providing a clean and soft backdrop that enhances readability and complements the primary hue.
- Accent color: A vibrant yet accessible medium blue (#4A96F5), analogous to the primary color, chosen to highlight interactive elements and call-to-actions, ensuring strong visual contrast.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean appearance and excellent readability across various screen sizes and text densities, suitable for both short headlines and informative paragraphs.
- Utilize a set of simple, clear, and modern line icons to represent key functions such as search, location, information, and directions, ensuring instant recognition and a clean aesthetic.
- Implement a clean, card-based layout for displaying registration centers, prioritizing mobile-first design. Key information should be immediately visible, with maps integrated fluidly into the user interface.
- Incorporate subtle and quick transitions for navigation and data loading states, such as smooth fades or gentle slide-ins, to provide responsive feedback without distracting the user.
